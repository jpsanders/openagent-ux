# Premium Color Palettes

## Purpose

**Avoid the generic SaaS palette.** Blue-500 and purple-to-pink gradients have become universal signs of AI-generated websites. These 10 curated palettes provide distinctive alternatives.

## CRITICAL: Never Use Blue/Purple

**Primary color NEITHER:**
- blue-500, blue-600 (any shade)
- purple (any gradient)
- "brand blue" without specific hue

**This is non-negotiable for premium uniqueness.**

## Never Use Palette

**NEVER use these:**
- Generic "corporate blue"
- blue-500, blue-600
- purple-to-pink gradient
- "primary blue" (too vague)
- Rainbow gradients

## Color Application Rules

### 60-30-10 Rule
- 60%: Primary (backgrounds, large areas)
- 30%: Secondary (cards, sections)
- 10%: Accent (CTAs, highlights)

### Dark Mode
- NOT just gray-900 (add tint)
- Use pure black (#000000) rarely
- Surface needs contrast
- Test all text legibility

### Accessibility
- Text: 4.5:1 minimum
- Large UI: 3:1 minimum
- Focus: 3:1 visible ring

## Quality Gates

Before marking build complete:
- [ ] Color: NOT blue-500 as primary
- [ ] Color: NOT purple gradient
- [ ] Palette: From curated list or brand
- [ ] Accent: 10% max coverage
- [ ] Contrast: 4.5:1 text
- [ ] Dark mode: tested
